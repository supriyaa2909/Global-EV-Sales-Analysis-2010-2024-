🚗 Global EV Sales Analysis (2010–2024)
###📊 Overview

This Power BI dashboard analyzes global electric vehicle (EV) trends from 2010 to 2024, focusing on sales growth, regional adoption, and powertrain insights.

###🎯 Key Insights
EV adoption increased rapidly after 2020
China leads global EV sales, followed by Europe and the USA
BEVs dominate over PHEVs in recent years
Passenger cars drive the majority of EV growth
EV growth is exponential, not linear
📁 Dashboard Pages
Regional Trends – EV growth by country/region
Powertrain Insights – BEV vs PHEV analysis
Parameter Analysis – Metrics like sales, stock, infrastructure
Time Series – Trends and comparisons over time
🛠 Tools Used
Power BI
Power Query
DAX

📌 Conclusion

EV adoption is accelerating globally, with BEVs and China driving the transition toward sustainable transportation.
